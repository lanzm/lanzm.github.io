## Docker常用指令 ##
- 安装Docker
Docker[官网](https://www.docker.com/)有详细的安装说明。
- 安装Docker compose
参照[官网](https://docs.docker.com/compose/install/)
- 查看已下载的镜像
```sh
docker images
```
- 查看正在运行的容器
```sh
docker ps
```
- 进入容器内部
```sh
docker exec -it xxx bash
```
- 停止容器及删除镜像
```sh
docker stop xxx / docker rmi xxx
```
- docker compose启动，停止以及查看容器运行
要 ``cd`` 到放置 ``docker-compose.yml`` 的文件下
```sh
docker-compose up / docker-compose down / docker-compose ps
```
- 编写自己的镜像
	1. 任意文件夹下创建一个文件夹，如
	```sh
	cd mnt 
	mdkir myfirst
	```
	``cd`` 到创建的文件夹下，创建 ``Dockerfile``
	```sh
	touch Dockerfile
	```
	2. 下面示例是编辑 ``tomcat`` 的 ``Dockerfile``
	```sh
	FROM daocloud.io/library/ubuntu:14.04
	MAINTAINER lan "943951063@qq.com"
	RUN apt-get update
	# copy (和dockerfile同一文件夹下) 容器中的位置
	COPY jdk1.8.0_111 /opt/jdk
	#ADD /mnt/software/jdk-8u111-linux-x64.tar.gz /opt/ 
	ENV JAVA_HOME /opt/jdk
	ENV JRE_HOME /opt/jdk/jre
	ENV PATH $JAVA_HOME/bin:$JRE_HOME/bin:$JAVA_HOME:$PATH
	ENV CLASSPATH .:$JAVA_HOME/lib:$JRE_HOME/lib:$CLASSPATH
	COPY apache-tomcat-7.0.77 /opt/tomcat
	#RUN unzip /usr/local/tomcat/apache-tomcat-8.0.33.zip 
	RUN chmod a+x /opt/tomcat/bin/startup.sh
	EXPOSE 8080
	ENTRYPOINT /opt/tomcat/bin/startup.sh && /bin/bash
	```
	3. 构建环境
	```sh
	#build –t (名称随便) (dockerfile所在的路径)
	docker build -t javaweb:0.1 /mnt/myfirst
	```
	4. 用 ``docker images`` 观察是否存在刚刚创建的镜像，最后跑起来，向外映射8081端口
	```sh
	#docker run -d -p 8081:8080 javaweb:0.1
	docker run -it --rm -p 8081:8080 tomcat:8.0
	```
- 用 ``docker-compose`` 构建 ``Tomcat + MySQL``，这里直接贴出 ``yml``
```sh
version: '3'  
services:  
    db:
        image: daocloud.io/library/mysql:latest
        restart: always
        environment:
            MYSQL_ROOT_PASSWORD: 123456
            #MYSQL_ROOT_HOST: 172.19.0.1
        volumes:
			# 本机路径：虚拟机路径
            - /E/docker/webTest/mysql/db:/var/lib/mysql
            #- /E/docker/webTest/mysql/my.cnf:/etc/mysql/my.cnf
    tomcat:    
       image: daocloud.io/library/tomcat:8.0.45-jre7  
       volumes:  
			# 本机开发的打包路径：虚拟机的tomcat路径
            - /F/IDEA/a/target/a.war:/usr/local/tomcat/webapps/a.war
       ports:  
            - "8880:8080"  
       restart: always
       links:
            - db
```
<font color="red">注意: </font>这里的数据库连接地址不是 ``localhost`` ，要进到容器内部输 ``cat /etc/hosts``查看容器地址。
